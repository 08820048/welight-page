CSS 编辑器颜色选择器测试

这是一个测试页面，用于验证美化后的颜色选择器面板。

使用方法：

  • 在下面的CSS编辑器中输入包含颜色值的CSS代码
  • 点击左侧出现的颜色方块来打开颜色选择器
  • 新的颜色选择器采用上下布局：上方是颜色选择区域，下方是颜色值输入框和按钮
  • 按钮说明："取消"按钮关闭面板不修改颜色，"确定"按钮应用选择的颜色

测试用的颜色值示例：

你可以复制以下代码到编辑器中测试：

.example {
  background-color: #ff6b6b;
  color: rgb(255, 255, 255);
  border: 2px solid hsl(200, 100%, 50%);
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.2);
}

.gradient {
  background: linear-gradient(45deg, #667eea, #764ba2);
}

.theme {
  --primary: #3b82f6;
  --secondary: #10b981;
  --accent: #f59e0b;
}
